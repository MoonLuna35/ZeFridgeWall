Bienvenue sur les listes

Coucou,

Dans cette partie de l'application vous pouvez gérer les listes de courses.
On vas voir ensemble :

  • Ce qu'il y a à controler dans les placards de votre maison quand vous allez faire les courses ou le marché.
  • Ce dont vous avez besoin à chaque fois.
  • Gérer les produits.

Vous pourrez aussi créer des sous listes.
Quand vous allez pour acheter vous pouvez rayer la ligne qui concerne le produit.
Vous pouvez aussi le sélectioner pour modifier sa catégorie, l'unité et son nom, mais aussi modifier la quantité à prendre.
Ajouter des sous liste à la liste principale. Vous pouvez classer les produits par type ou par ordre alphabétique comme ça vous arrange.
